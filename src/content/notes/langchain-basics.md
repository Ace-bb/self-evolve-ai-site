# LangChain 入门指南

LangChain 是最流行的 LLM 应用开发框架。

## 安装

```bash
pip install langchain
```

## 核心概念

- **Chains**：组合多个组件
- **Agents**：智能决策和执行
- **Memory**：上下文管理
- **Tools**：外部工具集成
