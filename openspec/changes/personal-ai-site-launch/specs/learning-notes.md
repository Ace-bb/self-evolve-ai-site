# Spec: 学习笔记模块

## Overview
技术学习笔记模块，支持笔记分类、知识整理、系列化学习路径。

## Requirements

### REQ-12: 笔记管理
- **描述**: Markdown 笔记，支持系列化组织
- **验收标准**:
  - 支持 frontmatter（标题、日期、分类、标签、系列、顺序）
  - 笔记按系列分组展示
  - 支持笔记内链接跳转
  - 笔记目录页（类似 Docusaurus sidebar）

### REQ-13: 分类与导航
- **描述**: 按技术领域分类笔记
- **验收标准**:
  - 分类列表：LLM、Agent、MLOps、前端、后端等
  - 树形分类导航（侧边栏）
  - 每个分类显示笔记数量
  - URL 路由: `/notes/category/:slug`

### REQ-14: 学习路径
- **描述**: 按系列组织的学习路径
- **验收标准**:
  - 学习路径页面（系列文章有序排列）
  - 进度指示（已完成/进行中/未开始）
  - 上一篇/下一篇导航
  - 路径概览卡片

### REQ-15: 代码与示例
- **描述**: 笔记中嵌入可交互代码示例
- **验收标准**:
  - 代码高亮
  - 代码复制按钮
  - 支持代码展开/折叠

### REQ-16: 搜索与过滤
- **描述**: 快速定位笔记内容
- **验收标准**:
  - 搜索框（复用博客搜索组件）
  - 按标签过滤
  - 按系列过滤
