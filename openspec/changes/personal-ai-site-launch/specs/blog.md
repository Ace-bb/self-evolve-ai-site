# Spec: 博客模块

## Overview
个人博客模块，支持 Markdown 文章发布、分类标签、全文搜索、RSS 订阅。

## Requirements

### REQ-1: 文章管理
- **描述**: 使用 Markdown 编写文章，自动解析为 HTML
- **验收标准**:
  - 支持 frontmatter 元数据（标题、日期、分类、标签、摘要、封面图）
  - 支持代码高亮（多种编程语言）
  - 支持数学公式（LaTeX）
  - 支持图片嵌入与懒加载
  - 自动生成文章目录（TOC）
  - 支持草稿状态（draft: true）

### REQ-2: 分类与标签
- **描述**: 文章可按分类和标签组织
- **验收标准**:
  - 分类列表页面，显示每个分类的文章数量
  - 标签云页面，按使用频率排序
  - 分类/标签过滤的文章列表页
  - URL 路由: `/blog/category/:slug`, `/blog/tag/:slug`

### REQ-3: 搜索
- **描述**: 全站全文搜索
- **验收标准**:
  - 搜索框位于导航栏
  - 支持标题、正文内容、标签匹配
  - 实时搜索建议（client-side search index）
  - 搜索结果高亮关键词

### REQ-4: RSS 订阅
- **描述**: 提供 RSS/Atom feed
- **验收标准**:
  - `/rss.xml` 输出标准 RSS 2.0
  - 包含最新文章（默认最近 50 篇）
  - 包含文章摘要和全文选项

### REQ-5: 文章列表与分页
- **描述**: 按时间倒序展示文章列表
- **验收标准**:
  - 每页显示 10 篇文章
  - 显示标题、摘要、日期、标签
  - 支持无限滚动或分页
  - 响应式布局（移动端友好）

### REQ-6: 阅读体验
- **描述**: 优秀的文章阅读体验
- **验收标准**:
  - 支持深色/浅色模式切换
  - 字体排版优化（中文友好）
  - 代码块复制按钮
  - 文章底部：上一篇/下一篇导航
  - 预估阅读时间显示

### REQ-7: SEO
- **描述**: 搜索引擎优化
- **验收标准**:
  - 每篇文章自动生成 OpenGraph / Twitter Card meta
  - sitemap.xml 自动生成
  - robots.txt 配置
  - 结构化数据（JSON-LD Article schema）
